## Blinkit Sales Data Analysis using Python

# Project Overview
This project focuses on exploratory data analysis (EDA) and visualization of Blinkit sales data using Python. The objective is to uncover meaningful business insights related to sales performance, outlet types, item categories, and location-based trends.

The analysis helps understand customer buying behavior, outlet performance, and sales distribution patterns, which can support business decision-making and strategic planning.

# Objectives

Analyze overall sales performance

Identify top-performing outlet types and locations

Understand product category contribution

Visualize key business metrics using charts

Extract actionable insights from raw sales data

# Tools & Technologies Used

Python

Pandas – Data cleaning & manipulation

Matplotlib & Seaborn – Data visualization

Google Colab

# Key Analysis Performed

Total sales by Outlet Location Type

Sales distribution across Outlet Types

Item category contribution to revenue

Sales trends and comparison analysis

Visual storytelling using bar charts and summary metrics
